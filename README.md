# BD_SQL

Evaluación de Módulo N°2 Base de Datos

Caso a Resolver:
Se busca entregar una solución adecuada a la problemática planteada en el enunciado del ejercicio, a través del diseño de un modelo entidad-relación, a partir del cual desarrollar una propuesta de Base de Datos Relacional, que permita registrar y gestionar información sobre un prototipo formativo y de evaluación de estudiantes. 

La secuencia de desarrollo de la solución propuesta considera la creación de:

* Modelo Lógico, creado con la herramienta Data Modeler de Oracle.
* Modelo Relacional, creado realizando Ingeniería al Modelo Lógico, utilizando Data Modeler de Oracle.
* Archivo DDL, generado en el mismo Data Modeler, a partir del Modelo Lógico/Relacional creado anteriormente, para ser posteriormente utilizado en SQL developer de Oracle.
* Scripts SQL que ingresen datos a la Base Datos Relacional.
*Scripts SQL que efectuen las Consultas para dar respuesta a las preguntas planteadas en la evaluación.

Se adjunta carpeta con archivos de respaldo, para poner a disposición una solución propuesta a los requerimientos solicitados,
